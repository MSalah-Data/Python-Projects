# 🚔 Crime Data Analysis Project

## 📌 Objective
Analyze crime data to uncover patterns in crime timing, location, and victim demographics.

---

## 💡 Problem Statement
Understanding crime patterns is essential for improving public safety. 
This project analyzes crime data to identify when and where crimes are most likely to occur, and which demographics are most affected.

---

## 📊 Key Insights

- 🕛 Peak crime hour: **12 PM**
- 🌙 Most dangerous area at night: **Central**
- 👥 Most affected age group: **26–34**
- 🧾 Most common crime: **Theft of Identity**
