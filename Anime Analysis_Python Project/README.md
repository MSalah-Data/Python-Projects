# 🎬 Anime Dataset Analysis

## 📌 Project Overview

This project analyzes an anime dataset to uncover insights about ratings, popularity, and content characteristics.
The goal is to transform raw data into meaningful insights using data analysis techniques.

---

## 🎯 Objectives

* Analyze the distribution of anime ratings
* Identify the most common and popular genres
* Explore the relationship between popularity and ratings
* Understand how features like episodes affect anime scores

---

## 📊 Dataset Description

The dataset includes information about various anime titles, with the following features:

* **Name** – Anime title
* **Genre** – Category of the anime
* **Type** – TV, Movie, OVA, etc.
* **Episodes** – Number of episodes
* **Rating** – Average user rating
* **Members** – Number of users (popularity indicator)

---

## 🧹 Data Cleaning

The dataset was cleaned to ensure accurate analysis:

* Removed missing values in key columns such as `rating`
* Converted `episodes` to numeric format
* Removed duplicate records
* Cleaned text fields for consistency

---

## 📊 Exploratory Data Analysis

Several analyses were performed to understand the dataset:

### 🔹 Rating Distribution

* Most anime ratings fall between **6 and 8**

### 🔹 Top Genres

* Genres like **Action** and **Comedy** are the most common

### 🔹 Popularity vs Rating

* No strong correlation between popularity and rating

### 🔹 Episodes vs Rating

* Number of episodes has limited impact on ratings

---

## 💡 Key Insights

* Most anime are rated متوسط إلى جيد (6–8 range)
* High popularity does not guarantee high ratings
* Some genres dominate the dataset
* Episode count has weak influence on ratings
